# Agents Overview

These guides orient AI coding agents to the major domains in this application. Review the subdirectories below for domain primers that link frontend and backend concerns.

## Directory Map
- `auth/AGENTS.md` – authentication flows, API endpoints, Pinia stores, and Vue pages.
- `teams/AGENTS.md` – team management, membership, usage limits, UI management surfaces.
- `invitations/AGENTS.md` – invitation lifecycle, tokens, acceptance flows.
- `prompts/AGENTS.md` – prompt creation, execution pipeline, visibility metrics.
- `responses/AGENTS.md` – response storage, term matching, visibility calculations.
- `organizations/AGENTS.md` – organization data, competitor tracking, and related UI.
- `articles/AGENTS.md` – article authoring, versioning, and chat integrations.
- `campaigns/AGENTS.md` – campaign aggregation, prompt batching, and reporting.

Start with the relevant domain file before diving into specific controllers, models, or Vue components.
