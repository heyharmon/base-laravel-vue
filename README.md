# LLM Mention Tracker

The application tracks how often specified terms appear in the responses of various Large Language Models (LLMs). Users can define prompts, run them across different providers, and collect responses to analyze term mentions. The stack consists of a Laravel API backend and a Vue 3 frontend.

## Future Enhancements
- Add support for additional LLM providers.
- Implement real-time alerts for term detections.
- Enhance analytics with more granular filtering and visualization options.
